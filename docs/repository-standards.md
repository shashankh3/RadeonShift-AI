# Repository Quality Standards

This checklist ensures the RadeonShift AI repository maintains an high-quality appearance for judges and future contributors.

## Pre-Launch Checklist
- [x] **README complete:** Includes architecture, business case, and quickstart.
- [x] **Screenshots added:** high-quality visual proof of the UI and PR scorecard.
- [x] **Secrets removed:** Checked for hardcoded API keys; `.env` is gitignored.
- [x] **Templates live:** Issue forms and PR templates are active in `.github`.
- [x] **License added:** Apache 2.0 License applied for open-core strategy.
- [ ] **Release tagged:** Cut a `v0.1.0-alpha` GitHub Release.
- [ ] **Repo topics added:** Tagged with `amd`, `mi300x`, `rocm`, `cuda`, `migration tool`, `llm`.
- [ ] **Actions passing:** CI/CD and bot workflows show a green checkmark.
